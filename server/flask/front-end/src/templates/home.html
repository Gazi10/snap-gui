About SUIEP

Snapchat User Interface Elements Pruner (SUIEP) is a tool for removing Snapchat UI elements from images. SUIEP uses object detection to identify UI elements in an image, and then inpaints where the identified elements are located in the image with a trained diffusion model.

Check out our Batch Clean-Up page to clean-up a batch of images with SUIEP, or personally select the elements you want to remove from your image on our Single Clean-Up page.
